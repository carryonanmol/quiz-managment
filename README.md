User Manual: Quiz Management System

Welcome to the Quiz Management System! This guide will help you understand and use the application, whether you are a student or an administrator.

What is This?

This is a desktop application that allows you to take quizzes or create them. It runs on a local network, meaning the server and the client applications must be on the same Wi-Fi or wired network to communicate.

Roles

There are two types of users:

Admin: A teacher or administrator who can create quizzes, add questions, and view the progress of all students.

Student: A user who can register for an account, take quizzes, and view their own past scores.

Quick Start: How to Use the App

To use the application, you just need to double-click the quiz_client.exe file. The login window will appear.

For Students

Here is how you can use the application as a student.

1. How to Register

If you are a new user, you must create an account.

On the login screen, click the "Register" button.

A new window will open.

Fill in your desired username and password.

Click "Register". If successful, the window will close.

You can now log in with your new account.

2. How to Log In

Enter your username and password.

Click the "Login" button.

3. The Student Dashboard

After logging in, you will see your dashboard. It shows you all the quizzes that are available to take.

4. How to Take a Quiz

Find the quiz you want to take in the list.

Click the "Start Quiz" button next to it.

A new window will open. A countdown timer will begin at the top of the screen.

Select your answer for each question and click "Next Question".

On the final question, the button will change to "Submit Quiz".

A message box will appear showing your final score.

Note: If the timer runs out, your quiz will be submitted automatically with the answers you have selected so far.

5. How to View Your Scores

On your dashboard, click the "View My Results" button. A new window will open showing a list of every quiz you have taken, your score, and the date you took it.

For Administrators

Here is how you can use the application as an administrator.

1. How to Log In

Username: admin

Password: admin123

Click the "Login" button.

2. The Admin Dashboard

You will see three main options:

Create New Quiz: Lets you create a new quiz.

Manage Quizzes: Lets you add questions to a quiz you already created.

View Student Progress: Lets you see the scores of all students.

3. How to Create a Quiz

Click "Create New Quiz".

Enter a Quiz Title (e.g., "History 101").

Enter a Time Limit in minutes (e.g., 15).

Click "Save Quiz".

4. How to Add Questions

Click "Manage Quizzes".

Find your quiz in the list and click the "Add/View Questions" button.

A new window will open. Fill in the question, all four options (A, B, C, D), and select the correct answer from the dropdown menu.

Click "Save Question". The fields will clear, allowing you to add another question.

5. How to View Student Progress

Click the "View Student Progress" button.

A new window will open, showing a complete list of every quiz taken by every student, their score, and the date.

Troubleshooting

"Connection Error": If you see a connection error, it means the client application cannot find the server. Please ensure the server application (quiz_server.py) is running and that both your computer and the server computer are on the same Wi-Fi network.
